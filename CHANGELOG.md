# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2024-02-18

### Changed

- Updated package name from `@linkio/react-native` to `@pt-nakul-sharma/linkio-react-native`
- Updated all documentation to reflect new package name
- Updated example app imports and dependencies
- Updated README badges and installation instructions
- Updated API documentation with correct package references

## [1.0.0] - 2024-02-18

### Added

- Initial release of LinkIO React Native SDK
- Deep linking support for iOS and Android
- Universal links and App Links handling
- Deferred deep linking support
- Referral tracking functionality
- TypeScript definitions
- Comprehensive API documentation
- Example application demonstrating all features
- iOS native module with Swift implementation
- Android native module with Kotlin implementation
- Event-based deep link listener
- Automatic pending link checking on app launch
- Manual deep link checking via API
- Custom metadata support for referral tracking
